# EXT Large Markdown Test Pack - With LaTeX

This pack contains five markdown files, each exactly 5 MiB uncompressed.

Purpose:
- Test large file loading
- Test preview rendering
- Test inline math
- Test display math
- Test matrices and aligned equations
- Test markdown tables containing formulas
- Test long single-line paragraphs with formulas

This version intentionally includes LaTeX examples for KaTeX or MathJax style rendering.
