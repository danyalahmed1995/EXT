# Broken Shell 5MB Examples

This pack contains intentionally malformed shell-script files totaling around 5MB uncompressed.

Use it to test that EXT:
- Opens malformed shell scripts as editable source text.
- Does not crash syntax highlighting, search, indexing, or navigation.
- Preserves broken source text unless the user edits it.
- Keeps large-file safeguards active.

These files are intentionally not meant to run.
