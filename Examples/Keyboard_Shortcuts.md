# Keyboard Shortcuts & Hotkeys

EXT is built for power users. Keep your hands on the keyboard and fly through your notes with these essential hotkeys:

## Global Navigation
- **`Ctrl` + `P`**: Quick Open / Search files in the active workspace
- **`Ctrl` + `W`**: Close current tab
- **`Ctrl` + `S`**: Save current file (EXT also auto-saves natively!)
- **`Ctrl` + `N`**: Create a new file in the active workspace
- **`Ctrl` + `Shift` + `N`**: Create a new folder

## Editor Controls
- **`Ctrl` + `B`**: Toggle **Bold** text
- **`Ctrl` + `I`**: Toggle *Italic* text
- **`Ctrl` + `K`**: Insert a Link
- **`Ctrl` + `Shift` + `C`**: Insert an inline code block
- **`Ctrl` + `Alt` + `C`**: Insert a multi-line code block

## Interface Toggles
- **`Ctrl` + `\\`**: Toggle Sidebar visibility
- **`F11`**: Toggle Fullscreen Mode
- **`Ctrl` + `,`**: Open Application Settings

*Note: On macOS, replace `Ctrl` with `Cmd`.*
